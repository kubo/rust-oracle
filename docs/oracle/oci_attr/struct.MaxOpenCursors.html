Struct oracle::oci_attr::MaxOpenCursors

source · 
pub struct MaxOpenCursors;
Expand description

A type parameter for Connection::oci_attr to get OCI_ATTR_MAX_OPEN_CURSORS, which is the maximum number of SQL statements that can be opened in one session

This returns the same value with the result of the following SQL statement when a database user has a privilege to access v$parameter.

select value from v$parameter where name = 'open_cursors'

Note that this attribute returns a proper value only when connected to a 12.1 server or later.
