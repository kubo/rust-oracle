Struct VarTypeMaxLenCompat

Source 
pub struct VarTypeMaxLenCompat;
Expand description

A type parameter for Connection::oci_attr to get OCI_ATTR_VARTYPE_MAXLEN_COMPAT as MaxStringSize. which controls the maximum size of VARCHAR2, NVARCHAR and RAW.

This corresponds to the result of the following SQL statement when a database user has a privilege to access v$parameter.

select value from v$parameter where name = 'max_string_size'
